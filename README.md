## Running ExpanseFlow page (test version)

Getting started: 
* open command promt in your local repository (files path)
* Run: py -m http.server 5500
* Open browser
* url: http://localhost:5500/

Requirements:
  * Python 3.12.4

!Please note that this is a local test version. Input data can be lost between sesions.!

## Workflow for this repo:
### Gitflow Workflow

+ __main__ conteains the releases files.

+ __Development__ it's the most advanced working copy (it contains all the completed Features merged toghether)

+ __Features__ are all improvements and modifications. All work in progress has to have a Feature branch open (may be unstable)
